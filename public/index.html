UML is the shared memory layer between your AI tools.

UML stores durable memory as pages, nodes, events, slices, edges, and semantic clusters. Connected tools save what matters, reinforce existing memories, avoid duplicates, and recall the right context later.

Connect UML in minutes.

Use the dashboard to create a private token for each tool, then paste the endpoint and token into that tool's connector or API settings.

1. Sign up or log in

Your memory belongs to your account.

2. Choose a tool

Use MCP for supported clients and API tokens for custom apps.

3. Generate a private token

Each connected app gets its own revocable token.

4. Paste endpoint and token

Add them in the tool's connector, MCP, tools, or API settings.

Keep tokens private

Never paste a live token into public chats, screenshots, repositories, or shared prompts. Revoke any token you no longer use.
